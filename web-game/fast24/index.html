Fast24

Make 24 before the timer gets away.

Time 0.0s
Difficulty Normal
Current expression
Select a number, an operator, then another number.

Rules

Use all four numbers exactly once to make 24. Click a number, an operator, and another number to combine them into a new value.

Addition, subtraction, multiplication, and division are allowed. Reset restores the current numbers without restarting the timer.

The timer stops when the final remaining value is 24.
